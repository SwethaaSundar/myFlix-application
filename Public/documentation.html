MyFlix Movies

Welcome to the Disney movies list!


Objective

To build the server-side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres.
Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Endpoint documentation

Data that endpoint returns HTTP Method Url Query parameters Request body Response
Return a list of ALL movies to the user GET /movies A JSON object containing a list of ALL movies.
Return data about a single movie by title to the user GET /movies/:title /:title A JSON object containing a single movie.
Return data about a genre by name/title GET /movies/genre/:gName /:gName A JSON object containing a specific genre.
Return data about a director by name GET /movies/director/:dName /:dName A JSON object containing a specific director.
Allow new users to register POST /users {
"name": "Joe",
"favMovies":[]
} 		A JSON object containing data about the new user.
Allow users to update their user info PUT /users/:Username /:Username {
"name": "Swethaa Sundar",
"favMovies":["Snow White and the 7 Dwarfs", "Moana"]
}		 A JSON object containing updated info about the user.
Allow users to add a movie to their list of favorites POST '/users/:Username/movies/:Movieid' /:Username/:Movieid Text message- Movie-title has been added to the users.
Allow users to remove a movie from their list of favorites DELETE /users/:Username/movies/:Movieid /:Username/:Movieid Text message- Movie-title has been removed to the users.
Allow existing users to deregister DELETE /users/:Username /:id Text message- User-name has been removed to the user list.
